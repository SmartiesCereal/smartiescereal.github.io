1. Download Forge 1.7.10

If you don't already have it, download and install the recommended installer for Forge for 1.7.10 here

If the installer won't open you may need to


2. Download Mods

Download all of the files from the google drive here.


3. App data

Go to your start menu search bar and type "%AppData%" this should open a folder in file explorer

4. Find .minecraft

Find a file named ".minecraft" in your app data folder

5. Mods Folder

Find and open the folder named mods

6. Copy Mods Over

Copy all of the mods that were downloaded in step 2 into the mods folder. If they are zipped up they should be extracted here. If done correctly, the contents of the google drive folder should be identical to the contents of the mods folder

7. Launch Minecraft

Open the minecraft launcher again and choose forge 1.7.10 as your version. Then press play and all the mods should have been installed!
