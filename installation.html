1. Source Forge

If you don't already have it, download and install sourceforge for 1.7.10 here.


2. Download Mods

Download all of the files from the google drive here.


3. App data

Go to your start menu search bar and type "%AppData%" this should open a folder in file explorer

4. Find .minecraft

Find a file named ".minecraft" in your app data folder

5. Mods Folder

Find and open the folder named mods

6. Copy Mods Over

Copy all of the mods that were downloaded in step 2 into the mods folder. If they are zipped up they should be extracted here.

7. Launch Minecraft

Launch minecraft java edition with the sourceforge 1.7.10 configuration and connect to the server!
